Personal Notebook
=================

This module allows users to attach personal notes to Omeka resources (item sets, items and media).

Personal notes are visible to its owner only.

Installation
------------

See general end user documentation for `Installing a module <http://omeka.org/s/docs/user-manual/modules/#installing-modules>`_
